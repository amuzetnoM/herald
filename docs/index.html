herald/connector/

MT5Connector: Manages MetaTrader 5 terminal connection with automatic reconnection, rate limiting, and health checks. Provides thread-safe access to market data and order execution.

herald/data/

DataLayer: Normalizes OHLCV data from MT5 into standardized pandas DataFrames. Calculates technical indicators (SMA, EMA, ATR, RSI) and handles multi-timeframe resampling.

herald/strategy/

Strategy (ABC): Abstract base class defining the strategy interface. Implementations like SmaCrossover generate Signal objects with entry/exit criteria and metadata.

herald/execution/

ExecutionEngine: Handles order submission with idempotency guarantees via client_tag tracking. Manages partial fills, position closure, and order status reconciliation.

herald/risk/

RiskManager: Approves or rejects signals based on position sizing rules, exposure limits, and daily loss thresholds. Provides emergency shutdown capability.

herald/persistence/

Database (Planned): SQLAlchemy-based ORM for storing signals, orders, trades, positions, and metrics. Supports historical analysis and performance tracking.

herald/indicators/ (Phase 2)

Indicator Library: Technical indicators for market analysis. Includes RSI (oversold/overbought), MACD (trend), Bollinger Bands (volatility), Stochastic (momentum), ADX (trend strength), ATR (volatility measurement).

herald/position/ (Phase 2)

Position Management: Tracks open positions with PositionInfo dataclass. PositionManager handles open/close/update operations, P&L calculation, and synchronization with MT5 terminal.

herald/exit/ (Phase 2)

Exit Strategies: Multi-strategy exit system with priority-based decision making. Includes StopLoss, TakeProfit, TrailingStop, and TimeBased exits. Returns ExitDecision with reasoning and metadata.

Example Strategy Usage Python 
from herald.connector import MT5Connector
from herald.data import DataLayer
from herald.strategy import SmaCrossover
from herald.risk import RiskManager
from herald.execution import ExecutionEngine

# Initialize components
connector = MT5Connector(config)
data_layer = DataLayer(connector)
strategy = SmaCrossover(config)
risk_mgr = RiskManager(config, connector)
executor = ExecutionEngine(connector)

# Main trading loop
while True:
    # Get market data
    df = data_layer.get_ohlcv("XAUUSD", "M15")
    
    # Generate signal
    signal = strategy.on_bar(df)
    
    if signal:
        # Risk approval
        if risk_mgr.approve(signal):
            # Execute trade
            result = executor.submit_order(signal)
            print(f"Order executed: {result}")
Database Schema SQL 
-- Signals table
CREATE TABLE signals (
    id INTEGER PRIMARY KEY,
    timestamp TEXT,
    symbol TEXT,
    signal_type TEXT,
    entry_price REAL,
    stop_loss REAL,
    take_profit REAL
);

-- Orders table
CREATE TABLE orders (
    id INTEGER PRIMARY KEY,
    signal_id INTEGER,
    order_type TEXT,
    status TEXT,
    filled_price REAL,
    FOREIGN KEY (signal_id) 
        REFERENCES signals(id)
);

-- Trades table with P&L
CREATE TABLE trades (
    id INTEGER PRIMARY KEY,
    order_id INTEGER,
    entry_time TEXT,
    exit_time TEXT,
    profit_loss REAL
);
